# 面试讲解模板

## 1. 项目背景
我做的是一个面向 A股投资者的信息监控系统，用来聚合热点题材、政策方向、重要经济数据、海外科技公司财报和全球产业会议。

## 2. 技术方案
前端使用 Next.js App Router、React、TypeScript 和 Tailwind CSS。后端使用 Next.js API Routes。第一版使用 Mock 数据，但接口层、类型层和数据库 SQL 都按照未来接入 Supabase/PostgreSQL 的方式设计。

## 3. 我的实现重点
我把系统拆成 Dashboard、热点题材页、事件日历页和后台管理页。热点页支持搜索、筛选和排序；日历页支持月份切换、市场筛选和事件详情弹窗；后台可以模拟新增、编辑和删除数据。

## 4. AI Coding 工作流
我先写 AGENTS.md 明确产品目标、技术栈、数据模型和开发阶段，再让 Codex 根据计划分阶段实现。开发中重点关注组件拆分、类型完整性、API 返回格式、数据库结构和 README 交付。

## 5. 后续扩展
后续可以接入 Supabase 持久化、真实数据采集任务、舆情评分模型、用户自选股订阅、定时任务和 Vercel 部署监控。
