# 后续可扩展功能清单

- 接入 Supabase，实现热点和事件的真实增删改查。
- 增加国家发改委、交易所公告、财联社等数据采集任务。
- 增加人工审核队列，避免自动采集内容直接影响展示质量。
- 增加自选股关注列表，按用户持仓或关注板块推送事件。
- 引入热度评分模型，结合来源权重、传播速度、相关股票数量和市场反馈。
- 增加 ECharts 或 TradingView Lightweight Charts，展示板块热度趋势。
- 增加定时任务和日志页面，展示采集成功率、失败原因和数据更新时间。
- 增加权限系统，区分普通访客、编辑员和管理员。
- 部署到 Vercel，并绑定 Supabase 环境变量。
