# 项目介绍文案

这是一个使用 AI Coding 工作流开发的 A股热点情报与全球事件日历系统，面向中国 A股投资者展示可能影响 A股、港股、美股和全球市场的重要信息。

项目第一版采用 Mock 数据，但完整设计了热点题材、市场事件、股票、板块、来源和关联关系的数据结构，并通过 Next.js API Routes 对外提供统一接口。前端页面包含 Dashboard、热点题材页、全球事件日历和后台管理页，视觉风格参考 Bloomberg Terminal、同花顺专业版和 TradingView Dark。

这个项目体现了前端页面开发、React 状态管理、TypeScript 类型建模、API 设计、数据清洗结构预留、Supabase/PostgreSQL 表设计、Vercel 部署和 GitHub 项目管理能力。
